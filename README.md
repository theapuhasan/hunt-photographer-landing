# Hunt Photography — Landing Page

Static marketing landing page for [huntphotographer.com](https://huntphotographer.com), the two-sided photography marketplace connecting clients with photographers.

Plain HTML/CSS/JS, no build step. Deployed to GitHub Pages via GitHub Actions (`.github/workflows/deploy.yml`) on every push to `main`.

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static file server.

## Custom domain

The `CNAME` file points this Pages deployment at `huntphotographer.com`. DNS must be configured to point at GitHub Pages separately (not managed from this repo).
